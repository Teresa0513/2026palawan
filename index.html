行程安排

桃園 ✈ 巴拉望 ➔ Tiki Bar

抵達公主港展開夢幻海島之旅。晚上前往 Tiki Bar 享受搖滾之夜與生力啤酒。

🍴 早餐：自理 / 午餐：自理 / 晚餐：生猛海鮮鍋(含龍蝦)
🏨 住宿：Princesa Garden Island Resort

與鯨鯊共舞 ➔ 雅士度假村

近距離觀察海洋溫柔巨獸——鯨鯊。隨後入住豪華 Astoria Palawan 度假村。

🍴 早餐：飯店內 / 午餐：海島野宴 / 晚餐：飯店內晚餐
🏨 住宿：Astoria Palawan Resort

巴頓港秘境跳島之旅

前往 Port Barton 跳島：珊瑚保護區尋海龜、下沉式沙灘、無人島浮潛。

🍴 早餐：飯店內 / 午餐：海島野宴 / 晚餐：飯店內晚餐
🏨 住宿：Astoria Palawan Resort

世界遺產地底河流 ➔ 滑索 ➔ 螢火蟲

探訪世界遺產地底河流，隨後體驗攀岩、高空滑索。晚上觀賞浪漫紅樹林螢火蟲。

🍴 早餐：飯店內 / 午餐：菲式自助餐 / 晚餐：Kalui 藝術家餐廳
🏨 住宿：Princesa Garden Island Resort

巴拉望 ✈ 台北

享受最後的熱帶早餐，帶著滿滿的回憶前往機場返回家園。

🍴 早餐：飯店內 / 午餐：機上或自理 / 晚餐：溫暖的家

貼心提醒

• 下水請務必穿著溯溪鞋/珊瑚鞋，保護足部不被礁石割傷。
• 鯨鯊行程嚴禁觸摸魚類，且禁止塗抹任何化學防曬乳。
• 建議攜帶薄外套，飯店與接駁車冷氣較強。

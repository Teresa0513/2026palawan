精彩行程安排

桃園 ✈ 巴拉望 ➔ Tiki Bar 搖滾之夜

飛往菲律賓最後的處女地——巴拉望公主港。抵達後展開驚奇之旅。

晚餐盛宴： 韓式生猛海鮮鍋，包含螃蟹、蝦、小卷、淡菜，加碼**每人一隻龍蝦(約200g)**及海戰車。

夜生活： 前往公主港最棒的 Tiki Bar 欣賞搖滾演奏，含門票及一瓶生力啤酒。

🍴 早餐：自理 / 午餐：自理 / 晚餐：韓式生猛海鮮鍋(含龍蝦)
🏨 住宿：Princesa Garden Island Resort 公主花園海島度假村
與鯨鯊共舞 ➔ 雅士度假村放鬆

搭乘螃蟹船尋找海洋巨獸——鯨鯊。您可以戴上蛙鏡下水，與全世界最大魚類近距離接觸。

下午入住 **Astoria Palawan**，享受私人海灘、水上樂園與壯麗的蘇祿海全景。

⚠️ 鯨鯊守則：禁止塗抹防曬油、禁止觸摸、禁止使用閃光燈、遠離尾部 4 公尺。
🍴 早餐：飯店內 / 午餐：海島野宴 / 晚餐：飯店內晚餐
🏨 住宿：Astoria Palawan Resort 雅士巴拉望度假村
巴頓港海洋公園秘境跳島

探索「小愛妮島」之稱的 **Port Barton**：

  • 【珊瑚保護區】尋找尼莫與海龜。
  • 【Paradise Island】在海灘樹影下發懶。
  • 【Luli Island】退潮才現身的下沉式沙灘。
  • 【Exotic Island】異國無人島，浮潛尋找彩色蝦。
🍴 早餐：飯店內 / 午餐：海島野宴 / 晚餐：飯店內晚餐
🏨 住宿：Astoria Palawan Resort 雅士巴拉望度假村
世界遺產地底河流 ➔ Zip Line ➔ 螢火蟲

探訪 UNESCO 世界自然遺產【地底河流】，搭乘人工划槳船觀賞壯麗鐘乳石。**本公司提供中文解說翻譯機**。

隨後挑戰 **Ugong Rock** 攀岩與 **Zip Line 高空滑索**。晚上深入紅樹林，見證螢火蟲閃爍的浪漫燈光秀。

⚠️ 地底河流每日限額 900 名入園，需提前提供護照影本預約。
🍴 早餐：飯店內 / 午餐：沙邦菲式自助餐 / 晚餐：Kalui 藝術家餐廳
🏨 住宿：Princesa Garden Island Resort 公主花園海島度假村
巴拉望 ✈ 台北

享受豐盛早餐後，前往機場搭機經馬尼拉返回台灣，結束愉快的巴拉望行程。

貼心服務： 全程無購物站，導遊車上提供特產選購（芒果乾、椰子油等）。

🍴 早餐：飯店內 / 午餐：自理 / 晚餐：機上餐點
🏨 住宿：溫暖的家

行前貼心提醒

  • ✔ 下水建議： 穿著溯溪鞋或珊瑚鞋以防割傷，身體保持水平漂浮。
  • ✔ 環保規定： 嚴禁觸摸海洋生物，下水請勿塗抹防曬乳。
  • ✔ 不可抗力： 若因天候導致地底河流無法入園，將退費 300 元或更換行程。

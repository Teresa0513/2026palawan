行程安排

桃園 ✈ 巴拉望 ➔ Tiki Bar

飛往菲律賓最後的處女地。抵達後展開驚奇之旅。

晚餐盛宴： 韓式生猛海鮮鍋，加碼 每人一隻龍蝦(約200g)。

🍴 早餐：自理 / 午餐：自理 / 晚餐：海鮮鍋(含龍蝦)
🏨 住宿：Princesa Garden Island Resort
與鯨鯊共舞 ➔ 雅士度假村

搭乘螃蟹船尋找海洋巨獸——鯨鯊。您可以下水與全世界最大魚類近距離接觸。

⚠️ 鯨鯊守則：禁止塗抹防曬油、禁止觸摸。
🍴 早餐：飯店內 / 午餐：海島野宴 / 晚餐：飯店內
🏨 住宿：Astoria Palawan Resort
巴頓港秘境跳島

探索「小愛妮島」之稱的 **Port Barton**：

  • 【珊瑚保護區】尋找尼莫與海龜。
  • 【Luli Island】退潮才現身的下沉式沙灘。
  • 【Exotic Island】異國無人島，浮潛尋找彩色蝦。
🍴 早餐：飯店內 / 午餐：海島野宴 / 晚餐：飯店內
🏨 住宿：Astoria Palawan Resort
地底河流 ➔ Zip Line ➔ 螢火蟲

探訪 UNESCO 世界遺產【地底河流】。隨後挑戰 **Ugong Rock** 攀岩與高空滑索。

🍴 早餐：飯店內 / 午餐：菲式自助餐 / 晚餐：Kalui 藝術家餐廳
🏨 住宿：Princesa Garden Island Resort
巴拉望 ✈ 台北

享受豐盛早餐後，帶著滿滿回憶前往機場返回台灣。

🍴 早餐：飯店內 / 午餐：自理 / 晚餐：機上餐點
🏨 住宿：溫暖的家

行前提醒

  • ✔ 下水： 穿著珊瑚鞋防割傷，保持水平漂浮。
  • ✔ 環保： 嚴禁觸摸生物，下水勿塗抹防曬乳。
  • ✔ 簽證： 請確保護照效期六個月以上。

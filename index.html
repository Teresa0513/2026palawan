每日行程安排

桃園 ✈ 巴拉望 ‧ 龍蝦盛宴
今日於桃園機場集合，帶著興奮的心情飛往菲律賓最後的處女地——巴拉望。抵達公主港後，迎接我們的是豐盛的【韓式生猛海鮮鍋】，包含螃蟹、海戰車，並特別加碼**每人一隻龍蝦**！晚間安排至全公主港最知名的 **Tiki Bar** 欣賞現場演奏，暢飲道地生力啤酒，為旅程拉開序幕。
🍴 用餐： 晚-韓式生猛海鮮鍋(含龍蝦)
🏨 住宿： Princesa Garden Island Resort 公主花園海島 SPA 度假村
與鯨鯊共舞 ‧ 五星渡假村
清晨搭乘螃蟹船尋找大自然的巨人——【鯨鯊】。與這種溫柔的巨型魚類共遊是人生難得的體驗（請遵守環保守則：不塗防曬、不觸摸）。下午入住 **Astoria Palawan Resort**，佔地 5 公頃的廣大園區內有私人海灘與水上樂園，讓您盡情放鬆或在蘇祿海邊漫步。
🍴 用餐： 早-飯店內 / 午-海島野宴 / 晚-飯店自助餐
🏨 住宿： Astoria Palawan Resort 雅士巴拉望度假村
巴頓港海洋公園跳島秘境
前往巴拉望隱藏版天堂【巴頓港 Port Barton】。這裡比愛妮島更加寧靜純粹。我們將前往【珊瑚保護區】浮潛尋找海龜與熱帶魚、在【Paradise Island】享受慵懶時光，並踏上隨潮汐隱現的【Luli Island】下沉式沙灘，感受與世隔絕的絕美海景。
🍴 用餐： 早-飯店內 / 午-海島野宴 / 晚-飯店自助餐
🏨 住宿： Astoria Palawan Resort 雅士巴拉望度假村
世界地底河流 ‧ 高空滑索 ‧ 螢火蟲
探訪 UNESCO 世界自然遺產【聖保羅地底河流】，搭船進入壯麗的石灰岩洞穴。下午挑戰 【Ugong Rock】 攀岩與超刺激的 【Zip Line 高空滑索】，從山頂俯視大片綠油油的田野。晚餐後前往河岸展開【螢火蟲之旅】，沉浸在河邊與林間閃爍的天然燈光秀。
🍴 用餐： 早-飯店內 / 午-沙邦菲式自助 / 晚-Kalui 藝術家餐廳
🏨 住宿： Princesa Garden Island Resort 公主花園海島 SPA 度假村
市區巡禮 ‧ 特產中心 ✈ 台北
最後一天悠閒享受飯店早餐與設施。隨後進行市區巡禮：參觀【Mitra's Ranch】觀景牧場、【Baker's Hill】麵包山，以及充滿異國風情的【聖母大教堂】。最後前往特產中心，採購著名的香蕉乾、芒果乾與椰子油伴手禮，帶著滿載的回憶前往機場，搭機返回溫暖的家。
🍴 用餐： 早-飯店內 / 午-市區自理 / 晚-機上餐點
🏨 住宿： 溫暖的家
